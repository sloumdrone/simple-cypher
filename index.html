Simple Cypher

Simple Cypher provides in and out for cypher text. you put in human readable text and Simple Cypher outputs cypher text. This cypher text can be decoded back into human readable text. At present this can be installed as python module for webserver usage or local computing. Providing low level encryption for password storage or other sensitive information you dont want lying around in your scripts. The version on the site here is meant to be a functional demo of the algorythm.

Working:

  • Cypher In
  • Cypher Out
  • Importable module based workflow
  • Web version

Not Working:

  • Support for carriage returns
  • Web version cannot support large amounts of text, due to utilizing get method
  • Bug where apostrophes in the cyphered/encoded text cause issues with Python output
